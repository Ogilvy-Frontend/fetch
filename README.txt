Instalando

1) shift + command + p
2) Digitar: Package Control: Install Package
3) Digitar:  Nettuts+ Fetch

2) shift + command + p
3) Digitar: Fetch: Manage
4) Selecioanar tudo e colar o conteúdo de fetch_manage.txt
5) Salvar
